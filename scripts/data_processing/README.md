# Data Processing Scripts

Corpus preparation helpers for L1/L2 data stages.

- `consolidate_corpus_l1.py` — collect/copy L1 sources into normalized layout
- `corpus_l1_to_l2_text.py` — convert L1 sources to L2 text outputs

Examples:

```bash
uv run python scripts/data_processing/consolidate_corpus_l1.py
uv run python scripts/data_processing/corpus_l1_to_l2_text.py
```
